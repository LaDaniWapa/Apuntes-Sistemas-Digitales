# Tema 05: Circuitos aritmético-lógicos

## Semisumador y sumador completo

- **Semisumador:**   Suma dos bits ($A$ y $B$). Genera una suma y un acarreo de salida

  - **NO** tiene en cuenta el acarreo de entrada
- **Sumador Completo**: Suma dos bits ($A$ y $B$) más una netrada de accareo $C$. Se usa para sumar números de más bits.

## Sumador Serie y Paralelo

Para sumar números de varios bits (ej. palabras de 4 bits), se pueden organizar los sumadores completos de dos formas:

- **Sumador Serie:**   Se usa un solo sumador y se van pasando los bits uno a uno (más lento).
- **Sumador Paralelo:**   Se utilizan tantos sumadores completos como bits tenga la palabra

> [!NOTE] ✏️ 
> Estos circuitos suelen operar utilizando el sistema de **Complemento a 2**, lo que permite usar el mismo hardware para sumar y restar.

## La Unidad Aritmético Lógica (ALU)

- **Concepto:**   Es un "super-circuito" que integra todos los anteriores (sumadores, restadores, multiplicadores, comparadores) y funciones lógicas (AND, OR, NOT) en un solo bloque.
- **Funcionamiento:**   Recibe los datos (operandos) y algunas **señales de control**. Dependiendo de qué señales de control activemos, la ALU "elige" qué operación realizar con esos datos y ofrece el resultado.
